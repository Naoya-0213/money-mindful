# 💰 がまん貯金（Money Mindful）

このリポジトリは、私 **野口直也（Naoya Noguchi）** が個人開発した節約支援アプリ「**がまん貯金（Money Mindful）**」のソースコードです。  
日常の“我慢した支出”を仮想的に貯金として記録し、節約を可視化・習慣化できることを目的としています。

## 💡 コンセプト

> 「我慢＝貯金」  
> 無理せず・楽しく・続けられる節約アプリ

日常の“ちょっと我慢した支出”を記録して、努力を可視化。  
「コンビニでジュースを買わなかった」「外食を控えて自炊した」など、  
小さな積み重ねを数字で実感できるように設計しました。

## 🌟 主な特徴

- **Supabase Auth** による安全なログイン / 登録機能  
- **目標金額・期限・目的** の設定機能  
- **我慢支出の記録（タイトル / 金額 / 日付 / カテゴリ / メモ）**  
- **グラフ表示（Chart.js）** による達成状況の可視化  
- **カレンダー表示（react-calendar）** による日別記録管理  
- **Zustand + React Hook Form** による効率的な状態管理  
- **モバイルファースト設計（レスポンシブ対応予定）**  
- **Vercel** によるデプロイ・ホスティング  
- **ESLint / Prettier** 導入でコード品質を統一

<br>

## 🔁 利用の流れ

1. **ユーザー登録（メール認証）**
2. **目標設定（目的・金額・期限）**
3. **我慢した支出を記録**
4. **履歴を確認・編集・削除**
5. **進捗をグラフで可視化**
6. **継続して記録 → 習慣化へ**

<br>

## 🧩 解決したい課題とそのアプローチ

| 課題 | 解決策 |
|------|--------|
| 節約の達成感が薄い | 節約額をグラフで可視化し、達成感を数値で実感 |
| 何を我慢したか忘れる | メモ・カテゴリ機能で振り返りやすく整理 |
| 継続が難しい | バッジ機能や進捗率でモチベーション維持（今後実装予定） |
